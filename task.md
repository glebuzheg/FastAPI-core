Контекст
В ходе выполнения тестового задания необходимо спроектировать и разработать core модуль для
FastAPI приложения, которое:
Основная задача разрабатываемого модуля - предоставление единых интерфейсов взаимодействия с
PostgreSQL и Redis для остальных модулей проекта, в рамках которых реализуется бизнес-логика.
Техническое задание
Реализовать модуль core со следующим функционалом:
Стэк
Предполагаемая структура проекта
Работает с PostgreSQL и Redis
Запускается как веб-сервер (Uvicorn)
Менеджмент зависимостей осуществляется через Poetry
Менеджер подключений к PostgreSQL (SQLAlchemy 2.0+ / async)
Менеджер подключений к Redis, поддержка async
Интерфейсы доступны через Dependency Injection
Использование lifespan
Конфигурирование через переменные окружения (Pydantic Settings)
Python 3.12+
FastAPI
Uvicorn
SQLAlchemy 2.0+ (с поддержкой async)
asyncpg
Redis (с поддержкой async)
Pydantic v2
Poetry
